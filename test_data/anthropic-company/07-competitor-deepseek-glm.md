# 竞品状况分析：DeepSeek + 智谱GLM

> 本文档对中国AI大模型领域的主要竞品进行分析。
> 当前时间：2026年7月

## DeepSeek（深度求索）

| 项目 | 内容 |
|------|------|
| 公司全称 | 杭州深度求索人工智能基础技术研究有限公司 |
| 成立时间 | 2023年 |
| 创始人 | 梁文锋 |
| 总部 | 中国杭州 |
| 员工规模 | 约500人（2026年Q2） |
| 核心产品 | DeepSeek系列模型、DeepSeek Chat |
| 最新模型 | DeepSeek V4 |

### 核心特点

1. **极致性价比**：DeepSeek模型以极低的训练成本著称，V2仅使用2000张GPU训练
2. **MoE架构先锋**：率先在开源模型中大规模应用Mixture-of-Experts
3. **开源友好**：模型权重和部分技术报告开源
4. **API低价**：API价格约为GPT-5的1/10，Claude的1/8

### 技术对比

| 维度 | DeepSeek V4 | Claude 4 Opus |
|------|------------|--------------|
| 架构 | MoE，671B总参（37B激活） | MoE，~200B总参 |
| 上下文 | 128K | 200K |
| 训练成本 | ~$500万 | ~$5亿+ |
| 推理速度 | 快（稀疏激活） | 中等 |
| 中文能力 | 极强 | 强 |
| 英文能力 | 强 | 极强 |
| 代码能力 | 强 | 更强 |
| 定价 | $0.5/$2（每M token） | $3/$15 |

### 竞争优势
- **成本优势**：训练和推理成本远低于美国公司
- **开源生态**：开源模型吸引大量开发者
- **中国市场**：不受对华禁令影响
- **快速迭代**：从V2到V4仅用一年半

### 竞争劣势
- **品牌认知**：国际知名度低于OpenAI/Anthropic
- **合规风险**：面临欧美AI监管不确定性
- **算力受限**：受芯片出口管制影响
- **生态规模**：API生态不如美国公司成熟

## 智谱AI（GLM系列）

| 项目 | 内容 |
|------|------|
| 公司全称 | 北京智谱华章科技有限公司 |
| 成立时间 | 2019年 |
| 创始人 | 唐杰（清华教授） |
| 总部 | 中国北京 |
| 员工规模 | 约800人（2026年Q2） |
| 最新模型 | GLM-5.2 |
| 核心产品 | ChatGLM、GLM API、CodeGeeX |

### 核心特点

1. **学术背景**：源自清华大学知识工程实验室
2. **GLM架构**：自研General Language Model，非标准GPT架构
3. **国产替代**：在中国政企市场有天然优势
4. **全栈布局**：从基础模型到应用产品的完整链路

### 技术对比

| 维度 | GLM-5.2 | Claude 4 Opus |
|------|---------|--------------|
| 上下文 | 1M tokens | 200K |
| 中文能力 | 极强 | 强 |
| 代码能力 | 强（CodeGeeX） | 更强 |
| 多模态 | 文本+图像 | 文本+图像 |
| 政企市场 | 绝对优势 | 无 |

### 竞争优势
- **政企市场**：中国政府部门和国有企业的首选
- **学术资源**：清华大学背景，人才和科研资源充沛
- **国产化**：完全自主可控的模型和技术栈
- **生态建设**：ModelScope等平台的深度合作

### 竞争劣势
- **国际市场**：基本没有国际影响力
- **创新能力**：更多是追随而非引领
- **算力瓶颈**：同样受芯片出口管制
- **商业化程度**：不及美国公司
